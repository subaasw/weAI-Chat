# WeAI Chat - Custom AI Chatbot System

**Tech Stack:** React, Shadcn UI, FastAPI, SQLModel, MySQL, ChromaDB, Crawl4AI

**Overview:**
Built a full-stack AI chatbot platform that learns from URLs and document files using embeddings and vector search.

**Key Features:**

- URL crawling with **Crawl4AI** and document ingestion

- **Embeddings + ChromaDB** for semantic search and retrieval

- Real-time chat streaming via **Server-Sent Events (SSE)**

- **Role-based authentication** (Admin/User) using JWT

- Auto chat title generation after multiple interactions

- User and admin dashboards with **MySQL** for management

- **SQLModel** for ORM and schema management

- Clean UI built with **React + Shadcn UI**
