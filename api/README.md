# API

## Installation 
Note: You have to install uv package manager for python 

### 1. create virtual environment
```bash
uv venv
```

### 2. Activate virtual environment
```bash
source .venv/bin/activate
```

### 3. Install Packages
```bash
uv sync
```

### 4. Crawl4AI Setup
```bash
crawl4ai-setup
```

### Run project
```bash
fastapi dev app/main.py
```
